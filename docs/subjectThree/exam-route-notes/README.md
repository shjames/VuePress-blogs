# 考试路线笔记

## 4号线

*   **起步**：50米内提速上二挡（起步前打左灯3秒后启动）。
    *   **注意**：起步一小段距离为实线，不能变道，不能压线。

*   **路口右转**：
    *   提前打右转向灯。
    *   车头盖住箭头时点刹。
    *   路口停车，退回一档。
    *   检查右转向灯是否亮，观察路口是否安全。
    *   确认安全后重新起步完成右转。

*   **右转后**：过实线后立马左变一条车道（紧接着做直线行驶），变道后立马摆正车身，保持二挡+20-25km/h。
    *   出来有一小段实线，尽量不要压线。
    *   打左转向灯。
    *   如有条件，直接上二挡。
    *   如无条件，过实线后直接停车重新起步（避免一档超50米）。
    *   观察安全后向左变一条车道。
    *   若是停车重新起步，等变道后再上二挡。

*   **直线行驶**：
    *   进入直线行驶区域前，提前使用二挡。
    *   保持20-25km/h的速度。
    *   把车身摆正（当前车在右二车道）。

*   **左掉头**：
    *   做完直线行驶后，下一个项目是左掉头（最左车道是掉头车道）。
    *   可提前变道到最左车道（共有4条车道，掉头位置会多出一条掉头车道）。
    *   **注意**：变道要分次进行，不能一次性变更多条车道。
    *   **注意**：直线行驶到掉头中途会经过路口和人行横道，记得点刹。
    *   过了第一个路口（人行横道）后是学校区域（无语音提醒），也要记得点刹。

*   **掉头区域操作（4号线无红绿灯）**：
    *   打左转向灯，进入掉头专用道。
    *   **情况一：前方有车挡住**
        *   先停车，等前车驶离。
        *   重新起步，为避免一档超50米，当肩膀对齐路口、需要打方向盘的位置后，再次停车，再重新起步。
    *   **情况二：前方无车挡住**
        *   直接使用二挡怠速（不踩油门）掉头，但要注意路口箭头处要点刹。
        *   方向盘往左打一圈，进入左二车道（务必转到此车道，因语音特定要求在此车道做超车项目）。
        *   掉头时务必注意右前方来车。
        *   进入左二车道后，若当前是一档，立即提速上二挡。

*   **超车项目**：
    *   **听到超车指令**：打左灯（同时看右后视镜）+看左后视镜+确认安全+3秒后，往左变一条道。
        *   如不安全，可停车等待。
    *   接着打右转向灯（同时看左后视镜）+看右后视镜+确认安全（不安全可停车等待，因有距离限制）+3秒后往右变回原车道。

*   **路口/人行横道**：
    *   前面经过路口+人行横道，要在箭头处点刹。

*   **变更车道**：
    *   操作同超车（注意打灯要满足三秒，尽量往左变道，即变到最左车道，因前面还有左转项目）。
    *   **注意**：若转向灯未自动回跳，要手动回灯。
    *   提前调直车身，准备做加减档。

*   **加减档项目（注意40码速度维持6秒）**：
    *   无语音提示，要主动做。
    *   **注意**：加油和刹车都要踩重点。
    *   尽量在最短距离做完加减档，避免路况不可预测性影响操作。

*   **公交车站/人行横道**：
    *   做完加减档后，看到前方高高的柱子（公交车站），要记得点刹（无语音提示）。
    *   一过白色柱子，地面会有标记点，车头盖住点就点刹。
    *   紧接着又会有一个人行横道，要记得点刹。
    *   **总结**：看到前方是人行横道、路口，车子盖住箭头时，都要点刹；转弯、掉头、会车时也要注意点刹。多点不扣分，少点就扣分，不确定是否点刹时，直接点刹即可。

*   **桥底左转**：
    *   经过桥底，项目是左转。
    *   听到语音“前方路口左转”，记得打左转向灯。
    *   因是路口，别忘记点刹。
    *   若左转过程中灯掉了，记得补回去。
    *   若是重新停车起步，在路口要完成上二挡，因距离较远，不上二挡会超50米。
    *   将车左转到左二车道。

*   **会车项目**：
    *   听到语音“会车”，点下点刹。
    *   紧接着又是一个路口+人行横道，再次点刹。
    *   若在前面的加减档项目中未完成，可过了此路口后再补一次加减档（当前在左二车道）。

*   **靠边停车准备**：
    *   看到右前方标有2km的蓝色牌子，可打右灯。
    *   车子过了牌子后，确认安全后，自主向右变一条车道（即变道最右边车道）。
    *   接着会看到限速30的白底红圈牌子和一个学校牌子（另一条线的），此时准备靠边停车。

*   **靠边停车操作**：
    *   **情况一：前方没有车挡住**
        *   可直接使用二挡怠速（不给油），降低车速靠边停车。
        *   **注意**：靠边停车速度一定要慢。
    *   **情况二：前方有车挡住**
        *   在车子进入靠边停车区域前停车等待。
        *   重新起步，使用一档左靠边停车。

*   **靠边停车注意事项**：
    *   靠边停车务必注意有没有打右转向灯。
    *   4号线靠边停车的开始区域是标有红色P字的电线杆位置。
    *   若要停车等待，切勿把车开超过该电线杆。
    *   电线杆和电线杆之间距离是37米。
    *   若是一档靠边停车，记得不能超50米。

*   **停车后**：
    *   记得回空挡，拉手刹（是否熄火根据实际情况决定）。

## 5号线

*   **起步**：注意要先打左转向灯。起步后立马加速上二挡。注意从起步到第一个人行横道都是实线，不能压线也不能变道。到达人行横道位置，车头盖住箭头要点刹。

*   **会车**：过了路口的下一个项目就是会车，记得点刹，注意左方的来车。

*   **超车项目**：过了路口后，正常情况下把车开到左二车道做超车项目（语音播放要求在左二车道做超车项目，视频中教练说如有特殊情况，也可在左三或左四，需练车时确认）。
    *   在做所有项目过程中，如发现不安全，都可停车等待，但注意不要停车在斑马线上（如超车、变更车道项目时，发现后方有来车，可停车等待，因超车有距离限制，无时间限制）。

*   **人行横道**：经过时记得点刹。

*   **桥底**：桥底有红绿灯和人行横道，要记得点刹或红灯就得停车等待。

*   **直线行驶**：过了人行横道就是做**直线行驶**项目，记得提前使用二挡，保持20-25km/h的速度，把车身摆正（现在车在左二车道，5号线这里只有两条车道）。

*   **人行横道**：又是一个人行横道，记得点刹。

*   **右转项目**：看到右边200m施工的牌子，就提前打右灯，确认安全后进入右转专用道（最右边车道）准备做右转项目。要注意看左前方有没有大货车，且这里没有红绿灯。**特别注意**：到路口时，如不停车，要记得在转弯处的“圆点”位置点刹，因这里的右转有一个人行横道，最后转入的是左二车道。

*   **公交车站**：紧接着会经过公交车站（注意地面上的圆点），要记得点刹。经过公交车站后这段路就只有两条车道了。

*   **人行横道**：又会经过一个人行横道，要点刹。

*   **变更车道**：提前打左灯，从左二变成左一。

*   **加减档项目**：接下来就是做加减档项目了（没有语音提醒）。

*   **人行横道**：经过时是红绿灯路口，红灯就得停车，绿灯就要点刹再通过。

*   **左掉头**：注意听到掉头的语音播放后，要记得打左灯。如是停车重新起步，要记得一档不能超50m，且要记得在肩膀对齐尽头时，要停下重新再起一次步，同时注意右前方来车。掉头的方向盘是左打一圈，然后进入到左二车道。
    *   这里调完头之后如果加减档没有做成功，可以再补一次。

*   **人行横道**：右经过人行横道（红绿灯路口），记得点刹或停车。

*   **学校**：经过学校，经过地面上的圆点记得点刹。

*   **左转弯项目**：如是绿灯直接通过，记得人行横道前点刹。方向盘打90度，不够再补。并且注意如是一档重新起步，在通过路口前要记得上二挡，左转进入右一车道，即最右车道。

*   **人行横道**：经过时点刹或停车。

*   **桥底**：经过桥底，又是一个红绿灯的人行横道路口，停车或点刹。

*   **靠边停车准备**：过了桥底就提前打好右灯，进入到最右边的车道准备做靠边停车。如需变多条，记得分多次变更车道。

*   **人行横道**：经过时记得点刹。

*   **靠边停车关键点**：要记住这个人行横道前有一个电线杆，若要停车等待，务必在此电线杆之前（即人行横道前）停车等待，因一过人行横道就是靠边停车区域了。还要注意靠边停车右转向灯务必亮起（5号线的靠边停车关键点就是要在最后的这个人行横道之前，务必把车身摆直）。

### 6 号线

*   **起步**：打左灯起步，50m内上二挡。若无机会，需停车重新起步。与4、5号线不同，此处为虚线，起步后可往左变一条车道。前方路口右转，教练视频中起步后自主左变一条道，但此处有两条道均可右转，是否不自主变道亦可？

*   **右转**：到达路口箭头处，先点一次刹车（右转），再停车。打右转向灯，确认安全后，起步右打一圈方向盘。转弯后是短距离实线，注意不能压线。过雪糕筒后，若右转向灯仍亮，记得回灯。随后需自主左变一条车道，记得打左转向灯。**注意**：前方是加减档项目，过雪糕筒后尽快左变车道。若无距离，停车等待机会。左变道后，摆直车身，准备加减档。

### 加减档 (无语音提示，需自主主动操作)

*   **路口人行横道**：经过时记得点刹或停车。过路口后可自主再左变更一条车道。

*   **学校路段**：路面圆点后记得点刹。

*   **变更车道**：提前打左灯，安全后往左变更一条车道（因下一个项目是左转弯）。此处三条道均可左转弯，也可视情况往右变一条道。

*   **左转弯**：路口箭头处点刹，注意右前方来车，转到左二车道。

*   **超车项目**：**注意语音提示**要求在右三（即左二）车道进行。左转弯后务必进入左二车道。**注意**：超车项目有两个语音提示，第一个“前方请从右侧三车道进入超车项目”非超车时机，无需理会；第二个“请完成超车动作后，再进行超车项目”才是。

*   **路口人行横道**：经过时记得点刹或停车。过路口后视频中要求自主往左变一条车道，原因待确认。

*   **会车项目**：前方会车项目，记得点刹。会车后，经过桥底，紧接着是人行横道，记得点刹或停车。**注意**：过桥底人行横道后，需提前摆直车身，因下一个项目是直线行驶。

### 直线行驶项目

*   **直线行驶完成**：立马又是一个人行横道，记得点刹或停车。

*   **左掉头项目**：听到“前方掉头”后，提前打左灯，进入掉头专用道（最左车道）。6号线掉头有红绿灯，务必注意是否为绿灯。若非绿灯，需重新停车起步。若为绿灯，路口时点刹一下（可询问教练绿灯是否可不停车直接走）。方向盘左打一圈，不够再补。掉头后进入最右车道（此段有2条车道）。

*   **人行横道**：经过时记得点刹。

*   **桥底**：有人行横道和红绿灯，记得点刹或停车。

*   **变道**：过桥底后打右灯自主向右变一条车道。**注意**：不要变道到最右车道，因最右车道是5号线终点（靠边停车区域）。

*   **人行横道**：经过时记得点刹或停车。

*   **5号线终点**：过5号线终点蓝色牌子后，打右灯自主往右变一条车道。

*   **前方路口右转**：提前打右转向灯。**注意**：箭头处和高处蓝色路牌下方地面圆点处均需点刹。右转时注意看右后视镜确认后方无来车。右转后务必处于右二车道（最右车道为非机动车道）。

*   **公交车站**：经过地面圆点时记得点刹。

*   **靠边停车**：最后一个项目。提前打右转向灯。**特别注意**：若需停车等待，务必在非机动车道尽头处之前停车等待（即非机动车道消失前）。过尽头即为靠边停车区域。非机动车道消失前，务必摆直车身方可进入靠边停车区域。靠边停车时务必保证右转向灯亮起，若发现不亮需补灯，并等待打灯5秒后再停车。
