# 考试路线笔记

### 4号线

- 起步后50米内提速上二挡（起步前要记得先打左灯3秒后才能启动）。
  - 注意：起步的一小段距离为实线，不能变道，不能压线。

- 路口右转：
  - 提前打右转向灯。
  - 到了路口箭头处（车头盖盖住箭头时）点刹。
  - 在路口停车，退回一档。
  - 检查右转向灯是否亮，观察路口是否安全。
  - 确认安全后重新起步完成右转。

- 右转后（要在过了实线后立马左变一条车道，因为紧接着要做直线行驶，并且要注意变完道后立马把车身摆正，保持二挡+20-25km/h的速度）：
  - 出来有一小段实线，尽量不要压线。
  - 打左转向灯。
  - 如果有条件，可以直接上二挡。
  - 如果没有条件，过了实线后直接停车重新起步（避免一档超50米）。
  - 观察安全后向左变一条车道。
  - 如果是停车重新起步的，等变道后再上二挡。

- 直线行驶：
  - 进入直线行驶区域前，提前使用二挡。
  - 保持20-25km/h的速度。
  - 把车身摆正（当前车在右二车道）。

- 左掉头：
  - 做完直线行驶后，下一个项目是左掉头（最左车道是掉头车道）。
  - 可以提前变道到最左车道（共有4条车道，掉头位置会多出一条掉头车道）。
  - 注意：变道要分次进行，不能一次性变更多条车道。
  - 注意：直线行驶到掉头中途会经过路口和人行横道，记得点刹。
  - 过了第一个路口（也就是人行横道）后是学校区域（无语音提醒），也要记得点刹。

- 掉头区域操作（4号线无红绿灯）：
  - 打左转向灯，进入掉头专用道。
  - **情况一：前方有车挡住**
    - 先停车，等前车驶离。
    - 重新起步，为避免一档超50米，当肩膀对齐路口、需要打方向盘的位置后，再次停车，再重新起步。
  - **情况二：前方无车挡住**
    - 直接使用二挡怠速（不踩油门）掉头，不过要注意路口箭头处要点刹。
    - 方向盘往左打一圈，进入左二车道（一定要转到这个车道，因为语音播放中，特定要求要在该车道做超车项目）。
    - 掉头时务必注意右前方来车。
    - 进入左二车道后，如果当前是一档，立即提速上二挡。

- 超车项目：
  - 听到超车指令：
    - 打左灯（同时看右后视镜）+看左后视镜+确认安全+3秒后，往左变一条道。
    - 如果不安全，可以停车等待。
  - 接着打右转向灯（同时看左后视镜）+看右后视镜+确认安全（不安全可停车等待，因为有距离限制）+3秒后往右变回原车道。

- 路口/人行横道：
  - 前面经过路口+人行横道，要在箭头处点刹。

- 变更车道：
  - 操作同超车（要注意打灯要满足三秒，尽量往左变道，也就是变到最左车道，因为前面还有左转的项目）。
  - 注意：如果转向灯没有自动回跳，要手动回灯。
  - 提前调直车身，准备做加减档。

- 加减档项目(要注意40码的速度要维持6秒)：
  - 没有语音提示，要主动做。
  - 注意：加油和刹车都要踩重点。
  - 尽量在最短的距离做完加减档，避免路况的不可预测性影响操作。

- 公交车站/人行横道：
  - 做完加减档后，看到前方高高的柱子（公交车站），要记得点刹（无语音提示）。
  - 一过了白色柱子，地面上会有标记点，车头盖住点就点刹。
  - 紧接着又会有一个人行横道，要记得点刹。
  - 总结：看到前方是人行横道、路口，车子盖住箭头时，都要点刹；转弯、掉头、会车时也要注意点刹。多点不扣分，少点就扣分，不确定是否点刹时，直接点刹即可。

- 桥底左转：
  - 经过桥底，项目是左转。
  - 听到语音“前方路口左转”，记得打左转向灯。
  - 因为是路口，别忘记点刹。
  - 如果左转过程中灯掉了，记得补回去。
  - 如果是重新停车起步的，在路口要完成上二挡，因为距离较远，不上二挡会超50米。
  - 将车左转到左二车道。

- 会车项目：
  - 听到语音“会车”，点下点刹。
  - 紧接着又是一个路口+人行横道，再次点刹。
  - 如果在前面的加减档项目中未完成，可以过了这个路口后再补一次加减档（当前在左二车道）。

- 靠边停车准备：
  - 看到右前方标有2km的蓝色牌子，可以打右灯。
  - 车子过了牌子后，确认安全后，自主向右变一条车道（也就是变道最右边车道）。
  - 接着会看到限速30的白底红圈牌子和一个学校牌子（另一条线的），此时准备靠边停车。

- 靠边停车操作：
  - **情况一：前方没有车挡住**
    - 可以直接使用二挡怠速（不给油），降低车速靠边停车。
    - 注意：靠边停车速度一定要慢。
  - **情况二：前方有车挡住**
    - 在车子进入靠边停车区域前停车等待。
    - 重新起步，使用一档左靠边停车。

- 靠边停车注意事项：
  - 靠边停车一定要注意有没有打右转向灯。
  - 4号线靠边停车的开始区域是标有红色P字的电线杆位置。
  - 如果要停车等待，千万别把车开超过该电线杆。
  - 电线杆和电线杆之间的距离是37米。
  - 如果是一档靠边停车，记得不能超50米。

- 停车后：
  - 记得回空挡，拉手刹（是否熄火根据实际情况决定）。

5号线

起步（注意要先打左转向灯），起步后要立马加速上二挡，还有注意从起步到第一个人行横道都是实线，不能压线也不能变道，到达人行横道位置，车头盖章箭头要点刹。

过了路口的下一个项目就是会车，记得点刹，注意左方的来车。

过了路口后，正常情况下把车开到左二车道做超车项目（语音播放是要求在左二车道做超车项目，视频中教练说如果有特殊情况，也可以在左三或者左四，这个要练车时确认一下）。

超车项目（在做所有的项目过程中，如果发现不安全，都可以停车等待，但是注意不要停车在斑马线上，比如在超车、变更车道项目时，发现后方有来车，可以停车等待，因为超车有距离限制，没有时间限制）。

经过人行横道，记得点刹。

经过桥底，桥底有红绿灯和人行横道，要记得点刹或者红灯就得停车等待。

过了人行横道就是做**直线行驶**项目，记得提前使用二挡，保持20-25km/h的速度，把车身摆正（现在车在左二车道，5号线这里只有两条车道）。

又是一个人行横道，记得点刹。

看到右边200m施工的牌子，就提前打右灯，确认安全后进入右转专用道（也就是最右边车道）准备做右转项目，要注意看左前方有没有大货车，还有就是这里是没有红绿灯的，还有特别注意的是，到路口的时候，如果不停车，要记得在转弯处的“圆点”位置点刹，因为这里的右转有一个人行横道，最后转入的是左二车道。

紧接着会经过公交车站（注意地面上的圆点），要记得点刹，经过公交车站后这段路就只有两条车道了。

又会经过一个人行横道，要点刹。

变更车道（提前打左灯，从左二变成左一）。

接下来就是做加减档项目了（没有语音提醒）。

经过人行横道（是红绿灯路口，红灯就得停车，绿灯就要点刹再通过）。

左掉头（注意听到掉头的语音播放后，要记得打左灯，如果是停车重新起步，要记得一档不能超50m，而且要记得在肩膀对其尽头的时候，要停下重新再起一次步，同时注意右前方来车），掉头的方向盘是左打一圈，然后进入到左二车道。

这里调完头之后如果加减档没有做成功，可以再补一次。

右经过人行横道（红绿灯路口），记得点刹或者停车。

经过学校，经过地面上的圆点记得点刹。

左转弯项目（如果是绿灯直接通过，记得人行横道前点刹），方向盘是打90度，不够再补，并且注意如果是一档重新起步的，在通过路口前要记得上二挡，左转进入右一车道，也就是最右车道。

经过人行横道（点刹或停车）。

经过桥底，又是一个红绿灯的人行横道路口，停车或者点刹。

过了桥底就提前打好右灯，进入到最右边的车道准备做靠边停车，如果是需要变多条，记得分多次变更车道。

经过人行横道，记得点刹。

要记住这个人行横道前有一个电线杠，如果要停车等待，就一定要在这个电线杠之前（也就是人行横道前）停车等待，因为一过了人行横道就是靠边停车区域了，还有要注意靠边停车右转向灯一定要亮（5号线的靠边停车的关键点就是要在最后的这个人行横道之前，一定要把车身摆直）。
