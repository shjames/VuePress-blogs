## 福州行程攻略

::: tip 交通提示
选择高铁站攻略：
- 福州站：靠近市区，交通便利
- 福州南站：距离市区较远，建议提前规划路线
:::

### 必游景点

#### 三坊七巷
::: tip 景点特色
福州最著名的历史文化街区，是闽都古城的缩影，也是绝佳的住宿选择。
:::

#### 福建西湖公园
::: tip 景点特色
福州市区内重要的人文景观，晨间可观看升旗仪式，感受庄严时刻。
:::
